WASINO JJ-1 Troubleshooting Guide

1. Verify Parameter Settings for New Servo Amplifier

  1. Locate Required Parameter Information:
    • Check the machine’s maintenance manual for the parameter tables (#1800–#1820, #1020, #3701).
    • Consult amplifier model specifications (A06B-6114-H206) to verify compatibility.
    • If available, use backed-up parameter data from the original amplifier as a reference.
  2. Check Parameters #1800–#1820 for Compatibility:
    • Access the parameter screen on the controller.
    • Verify each setting in the #1800–#1820 range, confirming they align with the original amplifier settings.
  3. Confirm Servo Communication and Spindle Settings:
    • Verify serial link settings in parameter #1020.
    • Check spindle-specific parameters (especially #3701, bit 4 for secondary spindle settings).
  4. Verify Axis-Specific Parameters with APZ Settings:
    • Ensure parameter 1815 (APZ) is set to 1 for each re-taught axis.
    • Power down and restart to confirm the APZ settings have saved correctly.

2. Diagnostic Display Check (No. 409)

  • Access Diagnostic Display No. 409 on the controller.
  • Observe the flags:
    • #3 (SPE): Should be 0. If 1, it indicates spindle parameter mismatches.
    • #2 (S2E): Should be 0. If 1, a fault with the second spindle.
    • #1 (S1E): Should be 0. If 1, a fault with the first spindle.
    • #0 (SHE): Should be 0. If 1, a serial communications module fault.
  • Take notes on any flags set to 1 for further troubleshooting.

3. Confirm APZ Re-Teach Settings

  • Verify parameter 1815 on the controller to confirm each axis has APZ set to 1.
  • Power down and restart to finalize settings.

4. Inspect Optical Cables and Connections

  • Visually inspect optical cables and re-seat each connector.
  • Test continuity if possible; replace cables if any faults are detected.

5. Check and Re-home Robot Arm (Parts Picker and Loader)

  • In manual jog mode, test the robot arm’s movement and re-home if needed.
  • Ensure it is clear of the spindle operational range.

6. Gather Reference Data from a Similar WASINO Machine

  1. Check Parameter Settings:
    • On a similar WASINO machine, go to parameter ranges #1800–#1820 and #1020, and note each setting.
    • Look for any differences or potential misalignments with your target machine.
  2. Observe Diagnostic Display (No. 409):
    • Review Diagnostic No. 409 for normal flags. This will serve as a baseline for troubleshooting on the problem machine.
  3. Check Amplifier and Hardware Configuration:
    • Verify that both machines use the same amplifier model (A06B-6114-H206).
    • Document any hardware or firmware version differences between the machines, as they may impact compatibility.
  4. Capture Reference Photos:
    • Take photos or screenshots of the parameter lists, diagnostic screens, and error logs (if allowed).
    • Use these images as a reference during troubleshooting.

Card Reader Instructions: Writing and Loading Parameters

Writing Parameters to a Memory Card

  1. Insert a memory card into the card reader, then insert the reader into the machine.
  2. Set the I/O Channel to 4 in the controller settings.
  3. Go to the SYSTEM menu > PARAMETER section, then choose WRITE or OUTPUT to save parameters to the card.
  4. Confirm the operation, then remove and verify the file on a computer if needed.

Loading Parameters from a Memory Card

  1. Insert the card into the target machine’s card reader.
  2. Set the I/O Channel to 4 in the settings.
  3. Go to SYSTEM > PARAMETER and select LOAD or INPUT to import parameters.
  4. Check key parameters after loading to ensure correct values.
