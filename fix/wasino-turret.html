Tecno Wasino JJ-1 Turret Zero Point Setting Procedure - FANUC 18i-TB

Procedure Overview

This procedure will guide you through resetting the zero point on the Tecno Wasino JJ-1’s servo-controlled turret using the FANUC 18i-TB controller. Follow each step precisely to avoid errors and ensure correct turret operation.

Steps

  1. Select the Turret Side:
    • Use the L/R select switch to choose the side you're working on:
    • L-side for the left turret
    • R-side for the right turret
  2. Switch to MDI Mode: Turn the operation mode switch to MDI.
  3. Enable Parameter Writing:
    • Press the OFFSET/SETTING key, then press [SETTING].
    • Set PARAMETER WRITE to 1 to enable parameter changes.
  4. Clear the Alarm:
    • The machine may display an alarm. Press RESET and CAN at the same time to clear it.
  5. Set the Turret Station Number:
    • Enter the number of turret head stations (typically 8) in the appropriate PMC counter address:
    • C40 for the left turret
    • C80 for the right turret
  6. Enter Maintenance Mode for the Selected Turret:
    • Set the correct PC parameter to 1:
    • K16.0 = 1 for the left turret
    • K16.1 = 1 for the right turret
  7. Unclamp the Turret Head:
    • Press the Reference Point Return button to prepare for unclamping.
    • Use the +Z jog button to fully unclamp the turret head.
  8. Adjust the Turret Position:
    • Use the +X or -X jog button to rotate the turret to station T1.
    • Rotate the turret 180° or more to ensure proper alignment.
  9. Clamp the Turret Head: Press the -Z jog button to securely clamp the turret head.
  10. Verify Clamping:
    • Check that the turret head is securely clamped.
    • If the turret does not clamp correctly, start over from Step 5 and repeat until clamping is correct.
  11. Power Cycle the CNC Unit:
    • Turn off the CNC unit power, wait 2-3 seconds, then turn it back on.
  12. Repeat Parameter Write and Clear Alarms:
    • Set PARAMETER WRITE to 1 again to enable parameter changes.
    • If an alarm reappears, press RESET and CAN together to clear it.
  13. Set Maintenance Mode Again:
    • Set the correct PC parameter to 1 again:
    • K16.0 = 1 for the left turret
    • K16.1 = 1 for the right turret
  14. Final Zero Point Confirmation:
    • Press the Reference Point Return button to ensure the turret is in sync with the controller.
    • Repeat the unclamp and clamp steps with the +Z and -Z jog buttons.
    • While holding down the -Z jog button, press the TURRET ON (indexing) button to confirm the zero point for station T1.
  15. Final Power Cycle:
    • Turn off the CNC unit again, wait 2-3 seconds, then turn it back on to finalize the zero point.
  16. Test Turret Indexing:
    • Test manual indexing by pressing the TURRET ON button.
    • For automatic indexing, use a T-code command in MDI mode to confirm correct movement.
    • Test both clockwise and counterclockwise indexing to ensure smooth movement and secure clamping at each position.
  17. Disable Parameter Writing:
    • If PARAMETER WRITE is still set to 1, set it back to 0 to lock the parameters.
    • If PARAMETER WRITE automatically resets to 0 after a reboot on your machine, this step can be ignored.
